Après avoir installer Python sur votre logiciel de développement, merci d'éxecuter ces trois ligne de commandes suivantes:


pip install pygame ---> Librairie Pygame
pip install pytmx ---> Permet d'utiliser les fichiers tmx générés avec Tiled, pour tout ce qui est asset de map
pip install pyscroll ---> Gère l'affichage de la map dans la fenêtre

Ensuite il suffit d'éexcuter le fichier main.py
